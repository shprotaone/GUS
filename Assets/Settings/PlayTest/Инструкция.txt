У тебя отдельная сцена PlayTest, в правом верхнем углу красный квадрат, значит это она.

Чтобы создать преднастройку уровня(скорость, высота и тд):
ПКМ => Create => LevelSettings
Или скопировать существующую настройку

Чтобы применить: в иерархии Boostrap => LevelSettings, туда перетягиваешь базовые настройки

Чтобы добавить их в очередь на след уровень:
---Systems--- => ProgressiveSystem, далее в parts жмем "+" в поле stageCount устанавливаем номер платформы  на которой будет применено изменение, в поле settings - закидываем настройку.

Все настройки сохранять строго в PlayTest

Если будешь делать префабы платформ, сохранять строго по пути Prefab => Sonic
Образцы текущих платформ по пути Prefab => Runner => Platforms
